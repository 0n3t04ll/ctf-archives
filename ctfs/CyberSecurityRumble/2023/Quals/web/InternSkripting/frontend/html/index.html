Coffee Flag Dispenser

In recent times coffee prices have been rising again. So we have to crack down on coffee purloiners using the coffee machine without paying for coffee!

So we introduce:

Coffee Flag

In order to get coffee from the coffee machine you now need a coffee flag. The flag changes each day and can be obtained by entering the correct secret into the dispenser. We then check that you are not a filthy coffee thief, to do so enter the secret the chief of coffee affairs has given you.
