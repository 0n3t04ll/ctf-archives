Factorization-因式分解

English:
Given an integer n with a length greater than 1e1200, write a Python function factorization(n: string) -> tuple[int]. The function aims to decompose n into the product of two prime numbers and return a tuple containing these two prime numbers. If there are multiple ways to decompose n, return any of them. If it is not possible to decompose n into the product of two prime numbers, return an empty tuple. Also, ensure that the algorithm completes within 5 seconds.

Ensure that the data range is within the condition: n > 1e1200.
