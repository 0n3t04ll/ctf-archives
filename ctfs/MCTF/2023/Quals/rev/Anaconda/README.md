RU:

Алекс, разработчик игр, создал змейку для игровых автоматов. Змея начала вести себя неадекватно, телепортируясь в другие автоматы и поедая змеек других игроков. Затем она переместилась на компьютер Алекса, готовясь съесть исходный код таска на M*CTF. Алекс поспешил по тревоге в свою комнату. Когда змея уже собиралась съесть исходный код, Алекс выключил компьютер. К сожалению, змея успешно съела исходный код, но, к счастью, оказалась заключена внутри компьютера..

EN:

Alex, a game developer, created a snake game for slot machines. The snake started behaving rogue, teleporting to other machines and consuming other players’ snakes. It then moved to Alex’s computer, preparing to eat the source code of an M*CTF challenge. Alex was alerted and rushed to his room. As the snake was about to consume the source code, Alex turned off his computer. Sadly the snake had successfully eaten the source code, but luckily was now trapped inside the computer..
